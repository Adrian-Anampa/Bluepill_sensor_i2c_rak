El siguiente archivo se utiliza de la siguiente manera :

1.Descarga todos los archivos subidos al git y asegurate que la carpeta tenga el nombre del archivo .ioc

2.Desde el STM32CUBEIDE abre la carpeta del proyecto

3.Realiza las conexiones correctas en el hardware y luego ejecuta el firmware en el MCU del STM32F103C8

4.Esperas que cargue el programa completamente y ya se podra utilizar para las pruebas que desees realizar
